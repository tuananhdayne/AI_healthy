# Scripts folder

Di chuyển các script tiện ích (ví dụ `build_faiss.py`, `load_data.py`) vào thư mục `scripts/` để làm gọn root.

Ví dụ cấu trúc đề xuất:

- scripts/
  - build_faiss.py
  - load_data.py
  - start_backend.sh (wrapper)

Sau khi di chuyển, cập nhật `README.md` để chỉ dẫn cách chạy các script.
